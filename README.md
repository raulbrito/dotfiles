dotfiles
========

my dotfiles

The approach used is to create a dotfiles/ directory in the home directory, and then create symlinks from the home directory.
This is the approach used by michaeljsmalley/dotfiles.

Adding new vim plugins as submodules can be done by following the instructions here: http://vimcasts.org/episodes/synchronizing-plugins-with-git-submodules-and-pathogen/
